# Advent of Code

Advent of Code challenges in c++

## Build

1. Run Build Script

```sh
./build.sh
```

## Test

1. Run Test Script

```sh
./test.sh
```

## Ubuntu Notes

install g++ for Ubuntu to get Neovim to work correctly

```sh
 sudo apt install g++-12
```

## Nvim Notes

run tests as a suite

```vim
<leader>a
```
